Descripción de los Algoritmos

El algoritmo Needleman-Wunsch realiza un alineamiento global de dos secuencias A y B. Fue propuesto en 1970 por Saul Needleman y Christian Wunsch. Es comúnmente usado para alinear secuencias de nucleótidos o proteínas. El algoritmo Needleman–Wunsch es un ejemplo de programación dinámica, y está garantizado que encuentre el alineamiento con el puntaje máximo.

El algoritmo Smith-Waterman es un algoritmo para realizar alineamientos locales de secuencias; determinar regiones similares entre dos secuencias de nucleótidos o proteínas. El algoritmo fue propuesto por Temple Smith y Michael Waterman en 1981. Smith-Waterman es un algoritmo de programación dinámica, posee la atractiva propiedad que garantiza encontrar el alineamiento local óptimo con respecto al sistema de puntaje que está siendo utilizado (que incluye la matriz de sustitución).

Basic Local Alignment Search Tool(BLAST)BLAST encuentra regiones de similitud entre secuencias biológicas. El programa compara secuencias de nucleótidos o proteínas con bases de datos de secuencias y calcula la significación estadística.

Un alineamiento múltiple de secuencias es un alineamiento de más de dos secuencias. Estas secuencias, como en el caso de los alieamientos por parejas pueden ser ADN, ARN o proteína. En todos los casos los algoritmos de alineamiento múltiple asumen que las secuencias que estamos alineando descienden de un antepasado común y lo que intentamos hacer es alinear las posiciones homólogas.

El modelo de Jukes y Cantor es un modelo que calcula la probabilidad de sustitución de un estado a otro. De este modelo también podemos derivar una fórmula para calcular la distancia entre 2 secuencias. La idea principal detrás de este modelo es la suposición de que la probabilidad de cambiar de un estado a otro diferente es siempre igual. Además, asumimos que los diferentes los sitios son independientes.

El modelo de dos parámetros de Kimura (K2P) (Kimura 1980 ) es probablemente el más utilizado de todos los modelos de sustitución de nucleótidos para estimar diferencias genéticas (generalmente llamadas distancias genéticas) y relaciones filogenéticas. No hace falta decir que los modelos precisos para la evolución de secuencias moleculares son muy importantes. Sin embargo, la razón por la que el modelo K2P se usa en exceso en estudios evolutivos y en estudios de códigos de barras de ADN no es porque el modelo K2P sea el modelo más preciso, sino probablemente porque muchos autores lo han usado, o porque es el predeterminado de varios paquetes para análisis filogenéticos.

UPGMA es un método de clustering (clasificación) que se basa en la identificación de las parejas más similares y en el cálculo de la media de las distancias entre ellas y el resto de las secuencias para reconstruir el árbol. Resulta conveniente usarlo solo si se cumple la hipótesis del reloj molecular (todas las secuencias han evolucionado a la misma velocidad) dado que genera árboles ultramétricos.
